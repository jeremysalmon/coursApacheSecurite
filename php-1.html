Installation du module PHP

Nous savons qu'un serveur Web ne sait que nous livrer des pages HTML (et les fichiers CSS, JS, Images, ...) associés.

Dans la majorité des cas un module nous permettant de développer des sites "dynamiques" est installé sur le serveur

Nous allons installer le module PHP pour Apache

Installez le module PHP pour Apache à l'aide de APT

Le module est maintenant installé et chargé dans Apache. La version CLI est également installée par défaut.

Le module Apache nous permet de faire des pages .php servies par le serveur Web

Le PHP Cli nous permet de faire des programmes en ligne de commande

Création d'un premier Script PHP

Créer un fichier /var/www/html/test.php et insérez le contenu suivant :

<h1>Ceci est mon premier programme PHP</h1>
<?php
 for($i=0 ; $i<100 ; $i++){
   echo("Bon Monsieur ".$i."<br>");
 }
?>

Connectez vous avec votre navigateur sur cette page pour voir le résultat

Création d'un script en ligne de commande

PHP nous permet également de faire des script en ligne de commande.

Créez un fichier /root/tail.php et insérez le contenu :

<?php
$parametre = $argv[1];
$monFichier = file('/var/log/apache2/access.log');
$nbLigne = count($monFichier);
$debut = $nbLigne - (int)$parametre;

for($i=$debut ; $i<$nbLigne ; $i++){
  echo("L".$i." : ".$monFichier[$i]."\n");
}
?>

Lancez votre programme : php tail.php 10

Modifiez le fichier PHP pour passer le nom du fichier à lire en paramètre

$parametre = $argv[2];
$monFichier = $argv[1];
$nbLigne = count($monFichier);
