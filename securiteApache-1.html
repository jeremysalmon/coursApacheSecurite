Cacher la version de Apache

Le premier élément de protection que nous allons mettre en place est de cacher la version de Apache dans les entêtes qu'il envoit.

En effet, par défaut, notre serveur Web envoit dans ses entêtes HTTP sa version et le système d'exploitation.

Avec l'extension Firebug dans Firefox par exemple vous pouvez voir les entêtes HTTP recues.

Editez à l'aide WinSCP / Sublime le fichier conf-available/security.conf

Deux paramètres sont à modifier :

  • ServerTokens Prod
  • ServerSignature Off

Vous devez redémarrer le serveur Web pour que le changement en compte (/etc/init.d/apache2 restart)

Les entêtes ne contiennent plus que Apache. La version et le système d'exploitation ne sont plus transmis.
