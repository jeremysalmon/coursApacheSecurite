Utilisation de WinSCP

WinSCP est un client SSH qui và vous permettre de parcourir le système de fichier de votre serveur Linux, directement depuis Windows à l'intérieur de fenêtres équivalentes à l'explorateur Windows.

WinSCP utilise le protocole SSH pour vous permettre de créer, modifier, supprimer fichiers et répertoires.

WinSCP, comme putty, se connecte à l'aide de vos identifiants Linux. Néanmoins il est a noté que pour travailler sur les fichiers de configuration présent par exemple dans /etc/ il faut être administrateur.

Par défaut, le serveur SSH est installé en interdisant la connexion en SSH avec le compte root. Nous allons modifié ce paramétrage pour pouvoir utiliser WinSCP correctement.

Lancez putty et connectez vous à l'adresse IP de votre machine Linux

Saisissez le compte utilisateur que vous avez créé au moment de l'installation de Debian.

Vous ête maintenant connecté à Linux depuis Windows

Passez en mode "Super Utilisateur" (root) en tapant su

Saisissez le mot de passe du compte root

Vous ête maintenant en root.

Editez avec nano (ou tout autre éditeur de texte que vous connaissez) le fichier /etc/ssh/sshd_config

Trouvez la directive de configuration PermitRootLogin et remplacez sa valeur actuelle par Yes

Pour que la modification soit prise en compte par le serveur SSH il faut le relancer le service.

Pour celà tapez dans la console de Linux /etc/init.d/ssh restart

Vous pouvez maintenant vous connecter en root à SSH

Lancez WinSCP et saisissez l'adresse ip, le login root et son mot de passe.

Vous serez alors connectés à votre machine Linux

Le panneau de gauche contient les fichiers de votre machine Windows. Le panneau de droite contient les fichiers de votre Machine Linux.

Vous n'avez plus qu'a configurer WinSCP pour utiliser Sublime Text comme éditeur par défaut.

Pour celà cliquez sur le menu "Options"/"Preferences"/"Editeur"

Cliquer le bouton "Ajouter" puis "Editeur Externe".

Sélectionner l'executable de Sublime Texte

Faites monter Sublime Texte en haut de la liste des éditeurs pour qu'il devienne l'éditeur par défaut.

Le panneau de gauche contient les fichiers de votre machine Windows. Le panneau de droite contient les fichiers de votre Machine Linux.
